# 🧠 AI Study Helper

## Description
AI Study Helper is a modern, lightweight, and responsive web application designed to simulate intelligent, AI-generated explanations for various study topics. It acts as an interactive academic companion, structuring complex subjects into definitions, practical examples, and bite-sized key points.

The app features a sleek, purely client-side architecture with built-in accessibility rules, gorgeous CSS micro-animations, and offline persistence.

## Features
- ✨ **Simulated AI Explanations**: Type in topics (e.g., "loops", "photosynthesis", "api") to receive dynamically generated, highly structured educational content.
- 💾 **Persistent Session Storage**: Click "Save Note" on any generated explanation to save it directly to your browser's native `localStorage`. It will remain there even when you refresh or close the page!
- 📚 **Interactive History**: View all your saved topics at a glance and click on any historical card to instantly reload it into the generator interface.
- 🌙 **Dark Mode Toggle**: A seamless theme switcher that intelligently remembers your preference and automatically respects your system's default color scheme.
- 🚀 **Accessibility-First Design**: Fully keyboard-navigable via robust focus rings, semantic HTML structure, and `aria-live` screen-reader broadcasts.

## How to Run
This application requires zero build steps, external dependencies, or backend frameworks. 

To run it locally:
1. Clone or download this repository to your local machine.
2. Open the project folder.
3. Simply double-click the `index.html` file to open it in your default modern web browser (Chrome, Firefox, Safari, Edge).
4. Start exploring and generating topics!

## Technologies Used
- **HTML5**: Semantic UI structure and accessibility rules.
- **Vanilla CSS3**: Highly modular styling containing native CSS variables, flexbox/grid layouts, advanced keyframe animations, and custom UI components.
- **Vanilla JavaScript (ES6)**: State management, DOM manipulation, asynchronous interface simulations, and offline `localStorage` handling. No external libraries required.
