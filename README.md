# 🧠 AI Study Helper

## Description
AI Study Helper is a modern, lightweight, and responsive web application designed to act as an interactive academic companion. By directly integrating with the Google Gemini API, it generates structured, highly polished educational content on demand, turning complex open-ended concepts into bite-sized key points and practical examples.

## Problem Statement
Students and self-learners often struggle to quickly grasp complex academic or technical topics. Traditional learning resources can be dense, overly theoretical, or poorly formatted, making it frustrating to find a simple definition or key takeaways when you need to understand concepts quickly.

## Solution
AI Study Helper solves this by providing structured explanations instantly. It uses generative AI to synthesize any topic into an intuitive learning card—extracting exactly what a learner needs: a plain-language definition, a concrete real-world analogy, specific code snippets (if applicable), and concise bullet points.

## Features
- ✨ **AI Explanations**: Dynamically generated, highly structured educational content powered by Google Gemini API.
- 💾 **Notes System**: Save generated explanations to a persistent library to review later. Click any historical card to instantly reload it.
- 🌙 **Dark/Light Mode**: A seamless theme switcher that intelligently remembers your preference for comfortable reading at any time.
- ⚡ **Fast UI**: A sleek, purely client-side architecture with built-in accessibility rules, gorgeous CSS animations, and instant loading from cache.

## Use Cases
- 🎯 **Exam Preparation:** Quickly generate structured summaries of confusing concepts right before an exam.
- 🔄 **Quick Revision:** Save and organize a personalized library of generated topics to easily review and refresh your memory over time.
- 💡 **Concept Clarity:** Whether you're a developer brushing up on an API or simply curious about a complex topic, get an immediate, easy-to-understand breakdown at the click of a button.

## How to Run
This application requires zero build steps, external dependencies, or backend frameworks. 

To run it locally:
1. Clone or download this repository to your local machine.
2. Open the project folder.
3. Simply double-click the `index.html` file to open it in your default modern web browser (Chrome, Firefox, Safari, Edge).
4. Start exploring and generating topics!

## Technologies Used
- **HTML5**: Semantic UI structure and accessibility rules.
- **Vanilla CSS3**: Highly modular styling containing native CSS variables, flexbox/grid layouts, advanced keyframe animations, and custom UI components.
- **Vanilla JavaScript (ES6)**: State management, DOM manipulation, asynchronous interface simulations, and offline `localStorage` handling. No external libraries required.
